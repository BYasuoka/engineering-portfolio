Senior Design / Research

Laser Microscopy Tissue Imaging

This project focused on making reflectance confocal microscopy more practical and accessible by pairing an affordable automated Z-stage with validation through interferometry and a computational pipeline for turning image stacks into 3D models.

Brenton Yasuoka, Alex Tocco, Anisha Kalla

Advisor: Dr. Jesse Wilson

Why It Matters

Noninvasive skin imaging

The poster frames the problem around skin cancer diagnosis and the cost and invasiveness of biopsy-based workflows, motivating a smaller and lower-cost RCM path.

Project Objectives

Three connected technical goals

Affordable automated Z-stage

Design and integrate a Z-staging mount capable of moving the sample through focus in a controlled, repeatable way for reflectance confocal microscopy.

Validation through interferometry

Build a reliable validation approach using a Michelson interferometer so the displacement behavior of the Z-stage can be checked against optical interference.

2D-to-3D reconstruction

Process image stacks with filtering, interpolation, and thresholding so OCT data can be rendered into a usable 3D surface and exported toward printable geometry.

Future handheld system

Use the project as a stepping stone toward a handheld, less expensive RCM platform for broader diagnostic access.

Methods

Z-stage and validation workflow

The poster shows an exciter-driven Z-stage concept paired with a Michelson interferometer validation setup. That pairing is important: the actuation system is only useful if the displacement can be measured and trusted at the scale required for optical focusing.

  • Exciter-based actuation for sample positioning
  • Interferometric fringe observation for displacement validation
  • RCM optical path integration with relay optics, wave plates, and galvanometer mirrors

Computation

Image stack to 3D model

The OCT workflow used contrast enhancement, non-local means filtering, thresholding, and stack interpolation to reconstruct a surface model from layered image data. That turns raw imaging output into something easier to inspect, compare, and eventually validate.

  • Raw OCT layers cleaned into processed image slices
  • Interpolated stack assembled into a continuous 3D surface
  • Result exported as an STL-style model for downstream use

Results

What the poster demonstrates

RCM operation after integration

The system remained operational after the new Z-stage and related modifications were integrated, which is a meaningful systems-level result for an optics-heavy platform.

Reference imaging captured

Images were collected from a U.S. Air Force target and a human hair sample, showing the microscope could still resolve straightforward reference specimens.

Interference fringes observed

Constructive and destructive interference patterns were captured, supporting the interferometer as a useful validation tool even though precise displacement verification was still unresolved.

3D reconstruction achieved

The team successfully reconstructed OCT image stacks into a printable 3D model, giving the project a strong computational output in addition to the optical hardware work.

Engineering Takeaways

  • The project spans mechanics, optics, validation, and image processing rather than a single isolated subsystem.
  • The work emphasizes feasibility: proving that an exciter-driven Z-stage can function in an RCM context.
  • Validation remains the hard part, which is often the reality in precision measurement projects.

Future Work

  • Resolve accurate Z-displacement verification for the exciter-driven stage.
  • Ethically source and validate skin sample images and corresponding 3D models.
  • Continue toward a handheld, commercializable RCM platform for inexpensive diagnosis.
